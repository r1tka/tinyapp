## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Clone repo (using the `git clone git@github.com:r1tka/tinyapp.git` command)
- Install all dependencies (using the `npm install` command).
- Run the development web server (using the `node express_server.js` command).
